# Intellego

Intellego is an AI-powered platform for learning Latin through reading, comprehension, pronunciation, and adaptive practice.

Unlike traditional Latin courses that focus heavily on memorization and grammar drills, Intellego prioritizes understanding. Learners progress through carefully structured content, receive immediate feedback, and build long-term retention using spaced repetition.

## Core Features

- Reading-based learning
- AI tutoring
- Adaptive exercises
- Pronunciation support
- Vocabulary mastery tracking
- Spaced repetition
- Progress analytics

## Vision

To become the most effective self-study platform for learning to read Latin.
